### Get todo by todoId

  `GET https://jsonplaceholder.typicode.com/todos/{todoId}`


### Get list todo

  `GET https://jsonplaceholder.typicode.com/todo`

### Add new todo

  `POST https://jsonplaceholder.typicode.com/todos` 

### Get todo by userId

  `GET https://jsonplaceholder.typicode.com/users/{todoId}/todos`
